Detailed Description

template<typename T, typename T_index = uint64_t, int compression_level = 3>
class CSF::SparseMatrix< T, T_index, compression_level >

Default class definition of the CSF::SparseMatrix class.

The CSF Sparse Matrix is a compressed sparse matrix format that is designed to be

optimized to real world redundant data with a strong negative binomial distribution.

Template Parameters
TThe value type of the matrix
T_indexThe index type of the matrix
compression_levelThe compression level of the matrix
