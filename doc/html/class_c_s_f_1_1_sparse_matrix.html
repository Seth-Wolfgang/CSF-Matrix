Detailed Description

template<typename T, typename indexT = uint64_t, uint8_t compressionLevel = 3, bool columnMajor = true>
class CSF::SparseMatrix< T, indexT, compressionLevel, columnMajor >

A class to represent a sparse matrix compressed in the CSF format.

CSF Matrix is a read-only sparse matrix class optimized for sparse-dense computation in cases where non-zero values are highly redundant. For such cases, sparse fiber storage can reduce memory footprint by up to 50% compared to standard sparse compression. CSF also increases the ability to further compress index arrays within each fiber. This default templated version is for compression levels 2 and 3 specifically. For compression level 1 there is a template specialization.
