Detailed Description

template<typename T, typename indexT, bool columnMajor>
class CSF::SparseMatrix< T, indexT, 1, columnMajor >

The CSF1 Sparse Matrix Class is a version of the CSC format and is the most basic version of the CSF format. Mostly used for ease of use with other compression levels and with other libraries. This is a good baseline to compare against the other compression levels as well.

Warning
This class does not have BLAS support like the other two compression levels. If one needs CSC BLAS support, Eigen is a good library to use.
