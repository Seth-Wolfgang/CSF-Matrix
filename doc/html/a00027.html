Detailed Description

template<typename T, typename indexT = uint64_t, uint8_t compressionLevel = 3, bool columnMajor = true>
class CSF::SparseMatrix< T, indexT, compressionLevel, columnMajor >

A class to represent a sparse matrix compressed in the CSF or CSC format.

Template Parameters
TThe data type of the values in the matrix
indexTThe data type of the indices in the matrix
compressionLevelThe compression level used
columnMajorWhether the matrix is stored in column major format
